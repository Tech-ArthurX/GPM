# GPM CLI

英文版: [README.en.md](README.en.md)

`gpm/` 目录负责包管理器运行时和命令行入口。

CLI 主要负责：

- 读取包索引。
- 解析包下载地址。
- 当元数据提供 `sha256` 时校验下载结果。
- 在本地展开 `.gpm` 包。
- 运行 GS 安装和卸载脚本。
- 对接 PE 工作流里需要的辅助工具。

## 安装模型

GPM 把包当作下载完成后的离线制品来处理。
CLI 先读取包索引，再下载 `.gpm` 文件，并在本地完成校验、解压和脚本执行。
服务端不需要参与安装过程，也不需要在线解析包内容。
